Is there any trial or free version of this plugin?
Yes, there is a free version available for this plugin. Kindly check the comparison table out and you will get every link there.

Can I use this plugin on multiple websites?
It depends on what license you choose during the purchase. The number of websites is mentioned with each license type.

Do I need to install any other plugin first?
No, this is a standalone plugin but if you want to show any specific item from any specific plugin then you need to install that plugin. For Example: if you want to show FlipBook then you need to install TNC FlipBook first.

Does this plugin conflicts with any other plugin?
We haven’t seen any such issues to date. This plugin is compatible with all the other plugins. But if you face any issue contact us through our support, and we will solve it for you.

What payment method do you accept?
We accept all types of credit cards, debit cards, Apple Pay, and Google Pay powered by Stripe.

How long will I get customer support?
You will get one year of customer support. For further support, you need to upgrade your support license.
