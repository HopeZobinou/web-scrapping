La mayoría de proveedores de alojamiento web optan por implementar cPanel, un estándar en la industria para la administración de sitios web, monitorizar su rendimiento y por realizar configuraciones en todos los niveles del entorno web. En cPanel, WordPress puede instalarse de forma sencilla y automatizada gracias a los instaladores de aplicaciones que se pueden implementar. En apenas unos minutos y sin la necesidad de poseer conocimientos técnicos se puede tener lista una instalación de WordPress.¿Qué es y para qué sirve?cPanel es un panel de administración para gestionar un servidor web a través de una interfaz completa, sencilla y amigable. La mayoría de hosting apuesta por cPanel para facilitar la administración de la web a sus clientes, pues no es necesario poseer conocimientos en desarrollo web o programación para utilizarlo.Cómo instalar WordPress a través de cPanelEn cPanel, muchos hostings implementan un instalador de aplicaciones con el que es posible instalar en muy poco tiempo y sin apenas esfuerzo, herramientas y CMS como WordPress, PrestaShop, Joomla o Drupal.Para realizar una instalación de WordPress utilizando cPanel hay que seguir los siguientes pasos:Acceder al instalador de aplicaciones de cPanel. Entre las distintas opciones que presenta cPanel al usuario se encuentra un instalador de aplicaciones. Es necesario localizar y acceder a esta opción para instalar WordPress desde cPanel.Seleccionar WordPress. Entre los distintos CMS y herramientas que ofrece el hosting, seleccionar WordPress para comenzar la instalación.Introducir los datos del sitio web. Al comenzar el proceso de instalación será necesario introducir algunos datos básicos para la creación del sitio web WordPress, como nombre del sitio, descripción, nombre del administrador y contraseña entre otros.Te puede interesar Cómo preparar el diseño de tu web pensando en SEOAlgunas dudas frecuentesReferente al uso de cPanel hay dos cuestiones que son las más solicitadas por los usuarios y que a continuación procedemos a resolver:¿Cómo acceder a WordPress desde cPanel?Para poder acceder a WordPress desde el propio cPanel y no visitando la web de login de administrador habitual (www.nombredediminio.com/wp-login.php), es necesario ir a la sección aplicaciones. Dentro de esta sección aparecerán las distintas plataformas que se hayan instalado en el servidor web a través de cPanel.Seleccionando el sitio WordPress se accederá a distintas opciones como realizar copias de seguridad, gestionar la base de datos MySQL, o iniciar sesión, entre otras. Seleccionando la opción Iniciar Sesión se podrá acceder a WordPress como administrador del sitio, y mediante la opción Visitar el Sitio, se podrá acceder al mismo como usuario.¿Cuál es mi usuario y contraseña de cPanel?Otra duda que suelen preguntar los usuarios de cPanel es cuál es el usuario y contraseña que deben utilizar para acceder al mismo. La solución a esta pregunta es muy simple, el usuario que se debe utilizar es el mismo que se utilizó cuando se crea la cuenta en el hosting, mientras que la contraseña es la que suministró el propio hosting vía correo electrónico, una vez realizada la contratación del servicio.Hay que tener en cuenta que, si se cambió la contraseña de acceso en algún momento, la contraseña original proporcionada por el hosting dejará de estar activa.cPanel es el administrador de sitios web más utilizado por los proveedores de alojamiento web debido a las facilidades y opciones que ofrece a los usuarios. Instalar WordPress de forma automatizada es una de las funciones más interesantes que se pueden realizar con cPanel, ahorrando mucho tiempo y esfuerzo a la hora de crear un sitio web con este CMS.
