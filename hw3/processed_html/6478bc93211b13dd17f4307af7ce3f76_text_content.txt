Sassio - SaaS Software & App WordPress Theme



































 
 
 
 
 
 
 
 


Sassio is a WordPress Theme for Startup, SaaS Business, App, Software, Agency, Seo, Digital Marketing, Payment Service, Business Solution, Social Marketing and more. It is suitable for you to demonstrate a professional approach to the technical services that you want to provide to people. Sassio is crafted with thorough understanding of the business, clean lines, soft colors, and fluid UX will surely create a unique website that your customers will love. Easy to use and customize, friendly with users.
Sassio comes with the 3rd version of the super powerful & customizable framework by GoodLayers. We have developed our theme’s core, gathered every feedback from customers, and put everything into this framework. It also comes with the most potent drag-drop page builder. It is super flexible with tons of great features such as background wrapper options, custom skin color for specific item wrapper, three view modes(live, preview, block), ajax page builder saving, pre-built pages and blocks and much more. Sassio is also bundled with the LightGallery plugin($18 value!) for free!


Core Features
- Schedule List in 5 main styles.
- Add Zoom(or other) icon in event list.
- Link your event to third party apps such as Zoom, Eventbrite, Hopin, etc.
- Speakers(personnel)
- Save $18 Plugins Value – included with LightGallery $18!
- Demo Importer – With a super powerful demo importer, you can have demo content on your site with a single click. We also have developed the multi-thread images importing technique to make it even faster!
- Powerful & Customizable Admin Panel
- Full / Boxed Layout
- 100% Fluid Layout – You can set 100% of the site’s content width to fit your screen size.
- Page items space can be customized – You can set space for each page item right in page builder setting
- Top Bar – You can choose to enable the top bar on top of all headers styles
- Top Bar Social – Enable social links in top bar easily
- 22 Header Layouts – With ability to control logo, navigation spaces.
- Custom Header – You can create custom header styles in each page.
- Secondary Navigation – You can choose the style of secondary navigation as slide style or overlay style. 
- Floating Navigation
- 6 Footer Layouts – You will also have the option to disable footer/copyright section in specific pages too.
- Google Fonts
- Unlimited Sidebars – Create sidebars and assign them to individual pages/posts easily.
- Choose sidebar layout from specific pages, posts
- Font Uploader – Upload your own font in EOT or TTF format
- Custom Skin Color – You can create custom skin color to use in the page builder wrapper. For example, if you have three sections of wrapper in a page, you can create skin and use them in each section.
- Create Thumbnail Size – allow you to create a custom thumbnail size to be used in portfolio, blog, gallery and in many other image elements.
- LightGallery – We provide two options for a lightbox for you. LightGallery and Strip.
- Additional CSS – Put your custom css easily via theme option. We also have a custom css box for mobile view too.
- Additional Javascript – Put your custom javascript easily.
- Maintenance / Coming Soon Page – Turn your site to maintenance or coming soon mode easily with a single click.
-  Import / Export  – Ability to import and export theme options, widget, custom page template easily.
- Post Format – 8 post formats – Standard, Aside, Image , Video, Quote, Link, Gallery, Audio
- Page Builder In Single Post – You can manage layouts freely in your single post with page builder
- Ability to set header background in single post
- Page Builder In Portfolio Single
- Ability to set header background in single portfolio
- Powerful Drag & Drop Page Builder
- Unlimited Color
- SEO Optimization
- WPML Fully Supported
- WooCommerce Fully Supported
- 16 Blog Layouts
- 19 Portfolio Layouts
- 6 Gallery Layouts
- Fully Responsive
- Automatic Theme Updates
- Live Customizer
