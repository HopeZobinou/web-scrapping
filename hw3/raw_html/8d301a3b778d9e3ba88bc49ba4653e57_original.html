Los 18 mejores plugins para WordPress GRATIS en 2023

Si estás buscando un listado con plugins gratis para WordPress, aquí tienes un listado con 18 mejores plugins WordPress gratuitos que permitirán mejorar los resultados de tu sitio web y que quizás no los estás utilizando. Este es mi Top con mejores plugins para Wordpress gratis.

WordPress es un CMS de código abierto que permite a desarrolladores de terceros poder crear infinitas herramientas, esa es la parte positiva.

La parte negativa es que al ver tantas opciones en el mercado muchas veces puedes sentirte perdido y no saber que plugin escoger para tu proyecto.

Por ese motivo, he recopilado 9 secciones importantes que puedes necesitar y en cada uno de ellos he recopilado dos plugins que merecen la pena que eches un vistazo.

Mejores plugins de seguridad WordPress sin coste

Jetpack: Plugin de seguridad para WordPress

Con 5 millones de instalaciones activas y sus 4 estrellas convierten a este plugin en un candidato para instalar en tu CMS de WordPress si quieres ganar seguridad en tu web, aparte de ofrecerte otras funciones como mejoras de rendimiento, marketing y diseño.

Este plugin gratuito de WordPress te ofrece copias de seguridad en tiempo real y la posibilidad de realizar restauraciones de manera sencilla. Este plugin es un veterano de la seguridad de las páginas WordPress.

Algunas características son:

  • Copias automáticas en tiempo real.
  • Restaurar tu sitio web en un solo clic.
  • Te ayuda a migrar tu contenido a un nuevo proveedor de alojamiento.
  • Si trabajas en equipo el mantenimiento de tu web, te permite saber que miembro ha ejecutado cada cambio.
  • Realiza amenazas de código o malware.
  • Protégete de formularios spam gracias a la colaboración con Akismet.
  • Te protege de los ataques de fuerza bruta.
  • Recibe notificaciones de cualquier aviso por correo electrónico.
  • Inicio de sesión seguro con la autentificación en dos factores.

En la sección de dudas frecuentes, el desarrollador afirma que las características base del plugin serán siempre gratuitas.

Wordfence Security: Plugin para proteger tu WordPress

Es un plugin freemium, por lo que podrás de disfrutar de el de manera gratuita, pero existen funciones que son de pago. Para la mayoría las funciones gratuitas son suficientes, por lo que no deberías de pasar por caja, aunque siempre tienes esa posibilidad si lo deseas.

Sus funciones principales son de hacer de cortafuegos y analizar el malware para que tu página web quede protegida.

Las funciones que puedes disfrutar de este plugin de seguridad son:

  1. Identifica y bloquea el tráfico que pueda resultar una amenaza para tu web.
  2. Te protege de los ataques de fuerza bruta limitando los intentos de inicio de sesión.
  3. Analiza los archivos y los plugins en busca de virus, puertas traseras, URL incorrectas, spam de SEO…
  4. Te avisa cuando un plugin ha sido abandonado o cerrado.
  5. Acceso a través de identificación en dos factores.
  6. Puede avisarte a través de SMS o por correo.
  7. Bloquea ataques realizados a través de IP.
  8. Impide el acceso a bots.

Más de 4 millones de usuarios activos están confianza en esta herramienta de seguridad para su WordPress.

Plugins Gratis de SEO (posicionamiento web) y caché

Wordpress Yoast Seo Free – posiblemente, el mejor plugin seo para wordpress

Yoast SEO gratis es uno de los mejores plugins de WordPress para ayudar a mejorar tu posicionamiento en los resultados de búsqueda de Google.

Nos permite corregir y optimizar las entradas del blog de acuerdo al SEO: palabras clave, titulo, meta descripción, url, etc.

Funciones destacadas:

  • Cuida automáticamente todo el SEO técnico de tu sitio;
  • Aporta información para mejorar el SEO de tu contenido web;
  • Optimiza la legibilidad del contenido;
  • Facilita la optimización para las redes sociales;
  • Permite crear sitemaps;
  • Identifica URL canónicas;

W3 Total Cache

W3 Total Cache es uno de los plugins más populares en la biblioteca de plugins de WordPress, es un plugin de caché para WordPress que deberías conocer.

Es un plugin open source, gratis. Con este plugin puedes puedes configurar todo lo relacionado con el caché y otros factores que mejoran la velocidad de carga de tu sitio web como la compresión, combinación y minificación de archivos (tanto CSS como Javascript), todos aspectos que ayudan a mejorar el SEO.

Algunas de sus funcionalidades:

  • Función minify (reduce el peso de los archivos) y combina varios en uno solo;
  • Soporte de servicios CDN;
  • Almacenamiento en caché de página;
  • Almacenamiento en el caché del navegador.

XML Sitemap Generator for Google

Este plugin crea automáticamente mapas de sitio XML para tu sitio web y notifica a los motores de búsqueda cada vez que se actualizan.

Es un plugin esencial para mejorar la estructura de tus páginas, haciendo que tu sitio de WordPress sea más fácil de encontrar.

Tiene funciones importantes como:

  • Indicará a los rastreadores cuáles son las URL más importantes de tu web de forma que indexarán lo que tú quieres.
  • Los robots de búsqueda no pierden tanto tiempo analizando una por una las URL de tu sitio web.
  • Los buscadores sabrán la frecuencia con la que actualizas tu web y registrarán rápidamente los cambios.

Mejores plugins para formularios WordPress

WPForms: Plugin para formularios de contacto

Este maquetador de formularios para WordPress tiene más de 5 millones de instalaciones y con una puntuación de 5 sobre 5.

Su facilidad de uso te permite crear formularios sin necesidad de saber programar o de escribir código web. Empieza a crear tu formulario de contacto con todas las secciones que deseas en cuestión de segundos gracias a la función de arrastrar y soltar.

Con esta herramienta puedes crear formularios de:

  • Comentarios.
  • Contacto.
  • Suscripción.
  • Pago.

¡Y todo ello en cuestión de minutos!

Las funciones principales de este plugin para formularios son:

  1. Genera formularios arrastrando y soltando.
  2. Plantillas prediseñadas.
  3. Adaptado para dispositivos móviles.
  4. Compatibilidad a nivel SEO sin afectar el rendimiento de carga.
  5. Sencillo de personalidad con tu imagen de marca.
  6. Posibilidad de incluir archivos, ya sea una imagen o un PDF.
  7. Recibes notificaciones por correo cada vez que alguien te escribe.

Plugin que cuenta con una versión gratuita y otra de pago.

Contact Form 7: Posiblemente, el mejor plugin de formularios para WordPress

Hay una palabra que define a las mil maravillas este plugin: Sencillo. Sus 4 estrellas sobre 5 valoradas por más de 1.900 usuarios lo avalan. Su interfaz es muy simple, pero te permite crear un formulario de contacto intuitivo de cara al usuario.

Algunas de sus funcionalidades y ventajas:

  • Es compatible con CAPTCHA;
  • Posee filtrados de SPAM;
  • Tienes la opción de personalizar el formulario;
  • Cuenta con una gran comunidad en Internet y puedes tener a mano documentación y vídeos que te expliquen paso a paso como poder tener tu formulario con este plugin gratuito.

Ideal para hacer tu primera caja de suscripción para el blog y empezar a crear una listad e contactos.

Mejores plugins WordPress de redes sociales gratuitos

Social Sharing Plugin – Sassy Social Share: plugin redes sociales WordPress

Como puedes apreciar en la imagen, es un plugin de WordPress gratis que está en constante actualización y que tiene una gran aceptación por parte de los usuarios.

Tener botones de medios sociales son importantes para ayudarte a la difusión de tus contenidos y que los usuarios te sigan en tus redes sociales. Sassy Social Share es consciente de ello y por eso te trae características tan importantes como:

  1. Iconos vectoriales y optimizados.
  2. Cumple con el RGPD.
  3. Más de 100 redes sociales a tu disposición.
  4. Iconos para que te sigan en redes sociales.
  5. Contadores de contenido compartido en redes sociales. Son compatibles con Twitter, Buffer, Pinterest, Reddit, entre otros.
  6. Personalización de iconos.
  7. Adaptado para móviles.
  8. Widgets y shorcodes.
  9. No necesitas registrarte.
  10. Tiempo de carga optimizado.
  11. Inserta los botones en diferentes lugares de tu web.

Simple Share Buttons Adder

El propio nombre ya lo dice, es un plugin sencillo de usar e instalar que cuenta con el apoyo de los desarrolladores a la hora de ir mejorándolo con el paso del tiempo y eso se ve reflejado en las valoraciones positivas que tiene.

Un buen plugin para agregar botones de tus redes sociales para que se pueda compartir y difundir el contenido de una manera sencilla.

Es compatible con muchos medios sociales como: Buffer, Digg, Pinterest, Reddit, Facebook, Twitter, LinkedIn, WhatsApp, email…

Las funciones que más me han gustado son:

  1. Elige las redes sociales que quieres son solo arrastrar y soltar.
  2. Puedes elegir diferentes modos de mostrar los botones, lo que te otorga un nivel de personalización.
  3. Agrega contadores sociales.
  4. Elige en que páginas quieres que se muestren los botones sociales.
  5. Fácil y sencillo de manejar.

Plugins de ecommerce para WordPress

WooCommerce

Es uno de los plugins de comercio electrónico más populares de WordPress. Te ofrece servicios gratuitos, aunque también tiene algunas funciones de pago. Con este plugin podrás convertir tu página web en una plataforma de comercio electrónico para poder empezar a vender tus productos.

Gracias a WooCommerce podrás realizar funciones como:

  1. Registrar tus productos.
  2. Instala un tema o plantilla acorde a tu marca.
  3. Elige los métodos de pago que prefieras con sus más de 100 pasarelas de pago a tu disposición.
  4. Configura las opciones de envío.
  5. Analiza el rendimiento de tu tienda.
  6. Conéctalo a tu tienda de Facebook.
  7. Es un plugin de código abierto.
  8. Haz remarketing con los carritos abandonados.
  9. Herramienta de promociones.
  10. Realiza informes para analizar el comportamiento de tu sitio web.

En palabras del desarrollador, es el plugin para WordPress más popular del mundo para crear un comercio electrónico.

FiboSearch – Ajax Search for WooCommerce: un plugin WordPress ecommerce

Este plugin te ayuda a tener una barra de búsqueda de productos más avanzada que la que ofrece la propia WooCommerce. La parte positiva de este plugin es que te va ofreciendo sugerencias en tiempo real según vas escribiendo el producto que quieres mirar.

Al facilitarle la vida al usuario, haces que la experiencia en tu web mejore, que puedan encontrar fácilmente los productos y, por lo tanto, las probabilidades de compra mejoren.

Algunas características importantes son:

  • Muestra el precio en los resultados de búsqueda.
  • Enseña la imagen del producto en el resultado de búsqueda.
  • Aparece la descripción del producto.
  • Adaptado a móviles.
  • Fácil instalación.
  • El usuario también puede buscar por categorías.
  • Puedes limitar el número de productos recomendados según la búsqueda.
  • Compatible con Google Analytics.
  • Personaliza la barra de búsqueda según tu identidad de marca.

Mejores plugins para cookies en WordPress

Cookie Notice & Compliance for GDPR/CCPA: plugin para política de cookies WordPress

Las cookies se convirtieron en un tema muy relevante con los últimos cambios de la Unión Europea por el tema de la privacidad de los usuarios y por la transparencia de las páginas web.

Para tener un buen plugin de cookies que cumpla con la normativa te recomiendo Cookie Notice, que te permite:

  • Modificar el texto del aviso.
  • Pedir consentimiento ya sea haciendo clic, scrool o cerrar.
  • Enlazar a la página de política de cookies para informar al usuario.
  • Personalizable en colores, tipo de botones y la posición de la barra de cookies en la web.
  • Elegir el tiempo de caducidad de las cookies.
  • No afecta al rendimiento SEO.
  • Compatible con WPML y Polylang.

CookieYes | GDPR Cookie Consent & Compliance Notice (CCPA Ready)

Sus 5 estrellas y más de un millón de usuarios activos están usando este plugin de cookies.

Aparte de cumplir con la normativa RGPD también acepta el cumplimiento de las normativas LGPD de Brasil, la CNIL de Francia y el CCPA de California.

Algunas de las características a mayores a tener en cuenta son:

  • Personalización a nivel estético.
  • Puedes colocar la barra de cookies en la parte superior o inferior de la web.
  • Opción de aceptar la cookie si el usuario hace scrool.
  • Cerrar la barra de cookies después de un tiempo establecido.
  • Compatible con qTranslate y Polylang.
  • Es apto con WPML.

Existen muchos más plugins de RGPD, pero estos sin duda son de los mejores plugins para cookies en WordPress.

Mejores plugins de comentarios WordPress gratis

Sistema nativo de comentarios de WordPress

Una buena opción es dejar el sistema que trae WordPress por defecto. Uno de los motivos es que al ser del propio CMS su integración y funcionalidad es total.

Ventajas:

  • Su diseño simple hace que el usuario sepa de manera intuitiva y clara que información introducir en cada campo y para que pueda comentar no necesita que se registre en ninguna plataforma.
  • Otro punto a favor, es que si quieres cambiar de plantilla o maquetador web no tendrás ningún problema y los comentarios de tu comunidad no se borrarán.
  • Dentro del propio WordPress puedes moderar los comentarios, aprobándolos o rechazándolos antes de que sean públicos y contestar a ellos desde la misma interfaz.

Comments – wpDiscuz

La funcionalidad de este plugin es la de mejorar los comentarios nativos de WordPress.

Mejora la velocidad de comentarios, además te ofrece notificaciones en tiempo real y posibilidad de dejar comentarios a través del inicio de sesión de numerosos medios sociales.

Características que me gustan de este plugin son:

  • Permite comentar de manera anónima de forma que no se pida ni el nombre ni el correo electrónico.
  • Los usuarios pueden editar sus comentarios.
  • Posee la función de <<Leer más>> para acortar los comentarios largos.
  • Puedes votar el comentario de forma positiva o negativa.
  • Puedes notificar al usuario que dejó el comentario de que su mensaje ya se aprobó.
  • Opción de fijar un comentario.

Mejores sliders plugins de WordPress gratis

Cuenta con más de 800.000 instalaciones activas y su autor Nextend está de manera constante actualizando el plugin, además de tener una buena crítica por parte de los usuarios con 5 estrellas de valoración.

Sus funcionalidades más destacables son:

  • Biblioteca de diapositivas.
  • Se integra bien con WordPress.
  • Diapositivas dinámicas.
  • Compatible con page builders como Elementor, DIVI, Visual Composer…
  • Diseño responsive.
  • Personalización de letras y estilos.
  • Optimizado para SEO.
  • Crea diferentes tipos de capas con: Imágenes, encabezados, textos, Vimeo, YouTube y botón.
  • Nueve animaciones de fondo diferentes.

Plugins para Adsense y anuncios Wordpress gratuitos

Ad Inserter – Ad Manager & Adsense ADS: para poner google adsense en mi web

Este plugin gratuito de publicidad para tu WordPress, desarrollado por Igor Funa, tiene una gran aceptación dentro de la comunidad con una valoración de 5 estrellas sobre 5.

Con este plugin de gestión de anuncios podrás insertar códigos de plataformas publicitarias tan conocidas como: Google Adsense, anuncios de shopping nativos de Amazon, Infolinks, etc.

Tiene una versión gratuita y luego si quieres características adicionales cuenta con una funcionalidad de pago.

Algunas características a tener en cuenta son:

  • Editor de resaltado de sintaxis.
  • Integración con AdSense.
  • Anuncios automáticos en entradas y páginas.
  • Insertar antes o des pues de; párrafo, contenido, entradas, imagen, comentarios, extracto…
  • Deshabilitar anuncios en entradas o páginas individuales.
  • Añade diferentes códigos en páginas AMP.
  • Editor ads.txt.
  • Compatibilidad con el RGPD.

AdRotate – Ad manager & Adsense ADS: plugin para anuncios

Con más de 40.000 instalaciones activas y con frecuentes actualizaciones, el plugin para anuncios de Arnan de Gans es un sólido candidato a ser tu plugin de ADS para tu WordPress.

Posee 4,5 estrellas sobre 5 y las numerosas funciones que te trae este administrador de anuncios para tu web te permitirá configurar de manera sencilla tus anuncios, ya sea con código HTML, JavaScript.

Al igual que en el caso anterior, también cuenta con una versión gratuita y de pago y está disponible en 6 idiomas, en los que se incluye el español, por lo que es una buena noticia.

Algunas de las funciones que puedes hacer con este plugin de anuncios es:

  1. Impedir que los bloqueadores de anuncios te eliminen el anuncio.
  2. Poder desactivar los anuncios una vez que caducan.
  3. Facilidad para insertar el anuncio en cualquier lugar de tu sitio web.
  4. Seguimiento de clics, impresiones y el CTR de tu publicidad en Adsense.
  5. Uso de códigos abreviados compatibles con tu página de WordPress.
  6. Avisos a tu correo electrónico o panel de control.
  7. Segmentación geográfica según el país y ciudad para anuncios en teléfonos móviles.

Sin cualquiera de los dos son buenas opciones para poner Google Adsense en mi web.
