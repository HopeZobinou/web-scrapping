Description

Thread Lock GP Blue is a medium strength, self curing adhesive for locking threads on most nuts, bolts and screws which may need to be disassembled.

Thread Lock GP Blue can be used in many areas including hydraulic system bolts, gearboxes, countersunk screws and most construction equipment.


  • Thixotropic non-drip formula.
  • Quick initial bond (5-10 minutes).
  • Can be used on “as received” and oil contaminated threads/nuts.
  • Medium strength for easy disassembly if necessary.
  • Gap filling up to 0.25mm.
