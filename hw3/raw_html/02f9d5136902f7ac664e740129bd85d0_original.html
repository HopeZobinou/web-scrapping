How to Create Image Marquee in Elementor

Learn how to craft an engaging image marquee within Elementor with ease, using Premium Addons Advanced Carousel widget. This article provides clear, step-by-step instructions, making it straightforward to create captivating image and video carousels that will add a dynamic flair to your Elementor-powered websites.

You’ll also find convenient links to access the essential tools, including Elementor, Premium Addons, and the widget itself, ensuring you have everything you need to get started on enhancing your web design skills right away.

Don’t miss out Elementor Advanced Carousel demo page!

Requirements:

  • You will need Premium Addons plugin installed and activated on your website.
  • Premium Addons works only with Elementor Page Builder plugin so you will need to have it installed and activated as well.
  • Add Elementor Advanced Carousel widget to your page by dragging and dropping it from Premium Addons’ widget list.

Create Image Marquee in Elementor

  • Add the images/videos you want to be added to the marquee by navigating to the widget settings -> Content tab -> then add your images/videos in the Items tab and adjust the items’ width, height, etc.
  • There are four animations Infinite, Flip, Flat, or Coverflow. To create the image marquee effect, make sure to set the carousel animation to Infinite. Adjust layout settings and you will have it all in action.

You Made It!

Now you have the images/videos Elementor Marquee using Premium Addons Advanced Carousel Widget.
